# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a respectful, harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive
- Giving and accepting feedback gracefully
- Focusing on what is best for the community and project
- Showing empathy toward others

Examples of unacceptable behavior:

- Harassment, intimidation, or discrimination
- Trolling, personal attacks, or insulting language
- Publishing others' private information without permission
- Any conduct that is inappropriate in a professional setting

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope

This Code of Conduct applies in project spaces (issues, PRs, discussions, and review comments), and when someone is officially representing the project in public spaces.

## Enforcement

If you experience or witness unacceptable behavior, report it via the security/contact channels listed in `SECURITY.md`.

All reports will be reviewed promptly and fairly. Maintainers may remove, edit, or reject comments/issues/PRs that violate this Code of Conduct.
